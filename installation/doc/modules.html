Installer YCP modules

This document gives a short overview which modules are provided by the yast2 packages. For detailed documentation, please look into the generated documentation for each file or use the source.
.

  • Installation
    Defines the installation environment.
  • Mode
    Defines the installation mode and flags.
  • Arch
    Defines the architecture of the target system.
  • Boot
    Handles all the boot stuff. i.e. which boot loader to use (LILO, SILO, MILO, ...), which kernel modules to put into the initrd, etc.
  • Language
    Contains language and encoding information.
  • Keyboard
    Defines keyboard type and layout.
  • Timezone
    Defines clock and timezone related information.
  • Mouse
    Defines mouse type and setup.
  • Console
    Handles the console stuff. The font, the encoding, etc.
  • Kernel
    Defines kernel and boot parameters.
  • X11
    X11 related data.

Some documentation about the modules API is also available.
